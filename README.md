Crypto Volatility & Correlation Analyzer

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://crypto-volatility-analyzer.streamlit.app/)

---

Business Context
In the DeFi (Decentralized Finance) ecosystem, effective risk management relies on understanding the covariance between primary assets. This project analyzes the correlation between BTC and ETH to identify high-volatility periods. These insights are critical for Liquidity Providers (LPs) to mitigate the risk of Impermanent Loss, ensuring more stable returns in automated market makers (AMMs).

Technical Stack
Python 3 (Google Colab)

Pandas: Time-series data processing, cleaning, and feature engineering.

Matplotlib & Seaborn: Data visualization, including scatter plots and return distributions.

yfinance: Real-time financial market data ingestion.
